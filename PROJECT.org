* Problem Statement

Traditional mechanic shops and rental car businesses in Gold Coast face:
- Manual booking (phone/WhatsApp) - lost customers
- No AI diagnosis - inefficiency
- Low fleet utilization - high costs
- Poor online presence - no visibility

* Solution

AI-powered system with:
- Instant diagnosis chatbot
- Online booking
- Fleet management
- Local SEO

* Architecture

#+BEGIN_SRC plantuml
@startuml
actor Customer
Customer -> Landing : Visit
Landing -> AI_Chatbot : Describe issue
AI_Chatbot -> OpenAI : Analyze
OpenAI -> AI_Chatbot : Diagnosis
AI_Chatbot -> Customer : Response + Quote
Customer -> Booking : Submit form
Booking -> Database : Save
Database -> Admin : Notify
@enduml
#+END_SRC
